# us-dev
### A user script development toolkit

## How to install
```bash
npm install https://git@github.com:TscCai/us-dev.git -g
# Solve the global dependencies
npm root -g

```

## How to use it
### Init project
```bash
mkdir test
cd test
usdev init test
```

### Build project
```bash
usdev build
```

### Deploy in browser
```bash
usdev deploy
```
